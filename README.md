# hey-cli

Ask your CLI, next command will be auto-generated.

## Usage

install it

```sh
cargo install hey-cli-app
```

ask it

```sh
hey show cpu usage on my mac
```

the next prompt will be auto-generated ðŸª„:

```sh
top -o cpu -s 5
```
